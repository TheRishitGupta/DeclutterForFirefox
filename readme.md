**Declutter – Chrome Tab Cleaner**

Declutter is a lightweight Chrome extension that helps keep your browser clean by closing unnecessary tabs on press of a button such as the Chrome new tab page and Google homepage. It works via a popup button or a keyboard shortcut and is designed to be safe, simple, and user-controlled.


**Features (Current Version 1.0.0)**

-Closes new chrome tab pages

-Closes Google homepage tabs (google.com, google.co.in)

-Works with one click from the extension popup

-Works with a keyboard shortcut

**Never closes:**
-The active tab
-Pinned tabs





**How to Install (Developer Mode)**

1.Open Chrome

2.Go to: chrome://extensions

3.Enable Developer mode (top right)

4.Download all files

5.Click Load unpacked

6.Select the declutter folder (root folder containing manifest.json)

The extension will appear in your toolbar



**Permissions Explained**

-tabs – Required to read and close tabs

-activeTab – Ensures safe interaction with the current tab

-storage – Reserved for future user preferences

-No unnecessary permissions are used.
